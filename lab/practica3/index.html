Redes locales

- Introducción -

Las redes de computadoras son una parte fundamental de la vida moderna, permitiendo la comunicación e interacción entre dispositivos. En esta práctica, se exploró el funcionamiento de una red de área local (LAN) utilizando switches y routers. El objetivo principal fue comprender, identificar y comprobar el funcionamiento de estos dispositivos en la creación de una red.

- Desarrollo -

1. Identificación y caracterización del switch y el router

Se comenzó por identificar las características físicas de ambos dispositivos, como puertos, indicadores LED y botones. Luego, se investigó su funcionamiento interno.

2. Creación de una red con switch

Se conectaron 3 computadoras y un servidor a un switch utilizando cables UTP de categoría 5e. Se configuró la asignación automática de IP y se verificó la conectividad entre los dispositivos mediante el comando ping. También se comprobó el acceso a la página web del servidor.

3. Creación de una red con router

Se conectaron 3 computadoras (2 alámbricamente y 1 inalámbricamente) a un router. Se configuró la asignación automática de IP y se verificó la conectividad entre los dispositivos y el servidor mediante el comando ping. También se comprobó el acceso a la página web del servidor

4. Unión de ambas redes

Se conectaron el switch y el router utilizando un cable UTP. Se verificó la conectividad entre los dispositivos de ambas redes mediante el comando ping y el intercambio de archivos.

    • - Conclusión -

    La práctica permitió comprender el funcionamiento de switches y routers en la creación de redes LAN. Se logró identificar los diferentes componentes de estos dispositivos, así como su papel en la transmisión de datos. Se comprobó la importancia de la configuración correcta de IP para la comunicación entre dispositivos.
